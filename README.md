 ## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Rekhaa9/smart-doc-assistant.git
   cd smart-doc-assistant
   ```

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Streamlit app:**
   ```bash
   streamlit run app.py
   ```
